# Drop your real avatar here

Place your actual Roblox-rendered avatar PNG at:

```
/public/images/avatars/uploads/avatar.png
```

The `AvatarCard` component will detect it and use it instead of the SVG fallback. Recommended specs:

- 460 × 575 (4:5) or larger, same aspect
- PNG with transparent OR pastel background
- Subject centered, facing camera, full-body
- 200 KB or less for fast loading

If the file is missing the site silently falls back to the hand-drawn SVG portrait — no broken image.
