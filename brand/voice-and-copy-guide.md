# Voice & Copy Guide — EMGamer731

## Voice principles

1. **Warm before clever.** Friendliness wins every coin-flip.
2. **Question-first.** When in doubt, open with a question — that's the slogan instinct.
3. **Speak to one person.** "You" never "you guys" in body copy. (Slogan is the exception, by design.)
4. **Never punch down.** No mocking gamer fails, no "lol u trash" energy.
5. **Always give a next move.** Every block ends with a way forward.
6. **Cute without sticky.** Sweet, not saccharine. We earn the smile.
7. **Inclusive defaults.** No assumed gender, school, region, or skill level.

## Tone matrix

| Surface | Tone |
|---|---|
| Hero | Inviting, slightly poetic |
| Live banner | Present, energetic, urgent without yelling |
| Watch / replays | Curatorial, "here's the good stuff" |
| Gallery | Generous, "browse forever" |
| Shop | Confident showcase, not desperate |
| About | Genuine, first-person from EMM |
| FAQ | Plainspoken, helpful, never legalistic |
| Errors / 404 | Cute, never blaming |

## Word choices

| Use | Don't use |
|---|---|
| Squad | Fam (overused) |
| Stream | Cast |
| Pull up a chair | Tap in |
| Replay | VOD |
| Drop | Launch |
| Cozy | Comfy |
| Morning | AM |
| The table | The community |
| Eat well, game better | "It's lit" |

## Forbidden

- Emoji-stuffed sentences (1 emoji max per sentence, used like punctuation)
- ALL CAPS in body
- Multiple exclamation marks
- "Don't miss out" / FOMO copy
- Gendered defaults ("girls" / "guys") — slogan exception aside
- Slang that ages out fast
- Comparative trash-talk vs. other creators

## Reusable phrases

- "Pull up a chair."
- "The table's set."
- "Stream's on."
- "We're live."
- "Pancakes loaded."
- "Today's drop."
- "Squad goals."
- "Dialed in."
- "Morning energy."
- "Eat with EMM."
